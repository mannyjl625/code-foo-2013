Wordsearch README
================

To complile program for running:

`````
javac Wordsearch.java
`````

To run:

`````
java Wordsearch
``````


Input
-----

For the wordsearch text file being passed in, the file must match the 
sample 'wordsearch.txt' in terms of format

`````
a	b	c

d	e	f

g	h 	i

Words to find:

hi
```
